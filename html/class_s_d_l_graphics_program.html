void SDLGraphicsProgram::loop ( )

Main loop that runs indefinitely.

Main loop of the SDLGraphicsProgram.

This function represents the main loop of the SDLGraphicsProgram. It continuously polls events from the event queue using SDL_PollEvent and handles the quit event to terminate the program. It also enables text input using SDL_StartTextInput and disables it before exiting the loop. The loop runs until the quit flag is set to true.
